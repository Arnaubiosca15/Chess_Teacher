# Installation

## Stable release

To install Chess Teacher, run this command in your terminal:

```sh
uv add Chess_teacher
```

Or if you prefer to use `pip`:

```sh
pip install Chess_teacher
```

## From source

The source files for Chess Teacher can be downloaded from the [Github repo](https://github.com/Arnaubiosca15/Chess_teacher).

You can either clone the public repository:

```sh
git clone git://github.com/Arnaubiosca15/Chess_teacher
```

Or download the [tarball](https://github.com/Arnaubiosca15/Chess_teacher/tarball/master):

```sh
curl -OJL https://github.com/Arnaubiosca15/Chess_teacher/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd Chess_teacher
uv pip install .
```
